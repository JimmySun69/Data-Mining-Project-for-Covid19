To run the dashboard, please install voila and voila gridstack.

To install voila and voila stack in jupyter notebook:

conda install -c conda-forge voila

conda install -c conda-forge voila-gridstack

voila --template=gridstack

To open the dashboard
-> Open File "SUN_RUITAO_SWE1709346_Dashboard"
-> Run the entire file
-> Click the Voila-dashboard to access the dashboard
